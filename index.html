version 0.8 changes:

Added a goblin temple. Replaced goblin mystics with goblin oracles. Dead goblins can no longer call the police. Cave trolls magically regenerate health, and drop rocks and fertilizer when killed.

version 0.7 changes:

Added a goblin legal system, an acorn-based economy, and theft.

Version 0.8 will include religious goblins. Version 0.9 will include Troll combat and spelunking.

version 0.6 changes:

You can no longer give gifts or otherwise interact with goblins after attacking them. Goblin lumberjacks will trade lumber for acorns. Trolls live in caves. Imprisoned goblins cannot defend themselves.

version 0.5 changes:

Added a goblin dice game and gambling debt. Added goblin tree goblin mystics and prophecies. Luck can impact goblin dice, gardening, and petting the farm cat.

version 0.4 changes:

Fixed a bug where petting the farm cat causes time travel. Tools can now be randomly crafted. The farm cat is uncomfortable around the altar. Added a goblin village and goblin bartenders.

version 0.3 changes:

The farm cat can be picked up and given snacks to eat. Tree goblins will be pleased if you give the farm cat as a gift.

version 0.2 changes:

Added the farm house

version 0.1 changes:

Added a title screen. Knocking on a tree no longer requires arms
